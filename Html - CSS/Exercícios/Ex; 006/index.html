Principais formatações de texto

Negrito ou destaque

Nessa frase temos um termo em negrito usando a tag < b > essa forma é não semântica.

Nesta frase temos um termo em negrito usando a tag < STRONG >.

Itálico

Nessa frase temos um termo em italico usando a forma NÃO semântica < i >.

Nesta frase usei o comando < cntrl + shift + p > para acessar uma aba e usar a "Wrab with abreviation" e apertei "i". É importante selecionar oq quer por itálico antes de entrar na aba de pesquisa.

Destaque

Nesta frase temos um termo em destaque usando a tag EM, o comando < cntrl + shift + p > é muito util para otimizar a escrita.

Texto marcado

Podemos criar também um texto marcado usando a tag Mark, para mudar a cor usei CSS com o comando style="background-color" dentro da tag Mark

Texto grande e texto pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.

Texto deletado

Podemos também marcar um texto como deletado "usei o marcador < del >", para mostrar que ele deve ser lido mas não inserido.

Texto inserido

Podemos marcar um texto como inserido "usei o marcador < ins >" para dar ênfase e indicar que ele foi adicionado depois.

Texto sobrescrito

Para escrever coisas como x20=4. Usei a tag < sup >.

Texto subscrito

Para coisas do tipo H2O1. Usei a tag < sub >.

Código fonte/Pré formatação

O comando document.getElementById (teste) é escrito em linguagem JavaScript. usei o comando < code > para melhorar o espassamento das letras no código.
